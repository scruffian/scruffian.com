Trowbridge Gallery

When I was at Trowbridge, one of my main roles was designing a new website with e-commerce functionality. We wanted to develop a range of different ways of ordering the products, so that people could search using different criteria. The search bar on the left gives you different ways of searchingg the site. The products are also linked to similar items.

This website was a lot of fun to work on. We introduced new features like choosing and changing the main displayed image and the frame it was displayed in, so the customer could customise their order.

I worked on this project from start to finish and also built a back-end ordering system.
http://www.trowbridgegallery.com

Alan Rogers .travel

This website selling good value camping holidays. The search facility is driven by XML which determines which campsites are available on the dates when you wish to travel. XML is also used to generate prices and book the campsite and ferries.

I also built a map search facility which integrates XML into Google Maps

Pink Scales

I designed this website for a music production company. I worked with Tom Stanley with some of the drawings. One of the main challenges of this website was that I never met the client - all of our communication was by email. Working like this means that you have to be very clear when you are describing ideas and experimenting with different designs. I think the final result is very pretty.

The site uses a content management system, which makes it easy to update.
